# Sabor Express

Sabor Express is a **simple Python console application** created as a learning project in the Alura Python course.

The goal of this project is to practice **Python fundamentals**, such as functions, conditionals, lists, and basic program flow, while building a small and realistic system.

---

## About the Project

Sabor Express simulates a small restaurant management system in the terminal. Through a menu, the user can interact with the application and perform basic actions.

This project was developed step by step during the course, focusing on code organization and readability.

---

## Features

* Display a main menu
* Register restaurants
* List registered restaurants
* Activate or deactivate restaurants
* Exit the application safely

---

## Technologies Used

* Python 3
* Standard Python libraries only

---

## How to Run the Project

1. Make sure Python 3 is installed on your machine
2. Clone this repository
3. Open a terminal in the project folder
4. Run the application with:

```bash
python app.py
```

The project is organized using functions to keep the code clean and easy to understand.
